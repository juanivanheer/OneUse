Bienvenido al servicio de reclamos de OneUse

A continuación se muestran los distintos tipos de reclamos con sus características principales. Utiliza luego aquel tipo que represente más al problema que tengas.

remove_shopping_cart
He pagado el alquiler, pero ya no estoy interesado en el objeto

El locatario tiene 12hs para cancelar el alquiler luego de haberlo pagado para obtener un reembolso. Pasadas las 12hs, se cancela el alquiler, pero se devuelve el dinero.

phone_disabled
No me puedo contactar con el Propietario

El dueño tiene 24hs para contactarse luego del alquiler. Si el reclamo se realiza antes de las 24hs, no se acepta ese reclamo ya que consideramos que el dueño aún está a tiempo de responder. Pasadas las 24hs, el reclamo se acepta y OneUse intentará contactarse con el propietario del objeto, avisando que el locatario no se puede contactar. Si no se puede establecer contacto en un lapso de 48hs, se realizara la devolución del dinero del alquiler al locatario.

image_not_supported
El producto que alquilé no estaba en las mismas condiciones que se mostraban en las fotos o no funciona correctamente

Si el propietario no ingresó el código proporcionado por el locatario, se procederá a devolver el dinero del alquiler al locatario. Si el propietario ingresó el código proporcionado por el locatario, no hay lugar a reclamo y el alquiler se cancela sin devolución del dinero.

speaker_notes_off
No me puedo contactar con el propietario para devolver el objeto

Se avisa al propietario que el locatario quiere devolverlo. Si a las 72hs no hay respuesta, el sitio liberara el dinero en garantía.

speaker_notes_off
No me puedo contactar con el locatario

El locatario tiene 24hs para contactarse luego del alquiler. Si el reclamo se realiza antes de las 24hs, no se acepta ese reclamo ya que el locatario aún puede responder. Pasadas las 24hs y si el reclamo se acepta, OneUse intentará contactarse con el locatario avisando que el propietario no se pudo contactar. Si no se puede contactar en un lapso de 48hs, se le otorga el dinero del alquiler al dueño.

report_problem
He tenido un problema con el objeto y no puedo cumplir

Se devuelve el dinero del alquiler al locatario y se cancela el alquiler

sync_problem
El locatario no se contactó conmigo para devolverme el objeto

Alguien del equipo de OneUse se contactará contigo para resolver el problema lo mas rapido posible

report
El objeto fue devuelto en malas condiciones

Alguien del equipo de OneUse se contactará contigo para resolver el problema lo más rápido posible
