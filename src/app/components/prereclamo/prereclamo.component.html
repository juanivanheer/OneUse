Bienvenido al servicio de Reclamos de OneUse

He pagado el Alquiler pero ya no estoy interesado en el objeto

El locatario tiene 12hs para cancelar el alquiler luego de haberlo pagado para obtener un reembolso, pasadas las 12hs, se cancela el alquiler pero se devuelve el dinero.

No me puedo contactar con el Propietario

El dueño tiene 24hs para contactarse luego del alquiler, si el reclamo se realiza antes de las 24hs, no se acepta ese reclamo ya que consideramos que el dueño aun esta a tiempo de responder. Pasadas las 24hs, el reclamo se acepta, OneUse intentara contactarse con el propietario del objeto, avisando que el locatario no se puede contactar, si no puede contactarse en un lapso de 48hs, se realizara la devolución del dinero del alquiler al locatario.

El producto que alquilé no estaba en las mismas condiciones que se mostraban en las fotos o no funciona correctamente

Si el propietario no ingreso el código proporcionado por el locatario se devuelve el dinero del alquiler al locatario. Si el propietario ingreso del código proporcionado por el locatario: No hay lugar a reclamo, el alquiler se cancela sin devolución del dinero.

No me puedo contactar con el propietario para devolver el objeto

Se avisa al propietario que el locatario quiere devolverlo. Si a las 72hs no hay respuesta el sitio liberara el dinero en garantía.

No me puedo contactar con el locatario

El locatario tiene 24hs para contactarse luego del alquiler, si el reclamo se realiza antes de las 24hs, no se acepta ese reclamo ya que el locatario aún puede responder. Pasadas las 24hs, el reclamo se acepta, OneUse intentara contactarse con el locatario avisando que el propietario no se pudo contactar, si no puede contactarse en un lapso de 48hs, se le otorga el dinero del alquiler al dueño.
